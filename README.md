# AIUB Portal+ ✨

AIUB Portal+ is a powerful and minimalist browser extension designed for AIUB students. It simplifies the login process by automatically solving math captchas and handling the login button click, so you can access your portal faster and with style.

---

## ✨ Key Features
- **Auto-Login:** Autometically logs you into the portal once credentials and captcha are filled.
- **Captcha Solver:** Integrated OCR engine to solve math problems instantly.
- **Modern UI:** A clean, minimalist interface that feels premium and responsive.
- **Dark Mode Support:** Fully compatible with dark mode for a better nighttime experience.
- **Personal API Key:** You can add your own OCR.space API key for personal use and higher reliability.
- **Cross-Browser:** Available for both Chromium-based browsers (Chrome, Brave, Edge) and Firefox.

## 🎥 How to Install
To install the extension manually on Chromium browsers:
1. Download the ZIP file.
2. Extract the ZIP folder.
3. Open `chrome://extensions/` in your browser.
4. Enable **Developer Mode** (top right).
5. Click **Load unpacked** and select the extracted folder.

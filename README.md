# AIUB Portal+
A minimalist browser extension for AIUB students that automates logins in portal instantly.

---

## Key Features
- **Auto-Login:** Instant entry once credentials and captcha are filled.
- **Captcha Solver:** Integrated OCR engine solves math problems in real-time.
- **Modern UI:** Premium minimalist design with full dark mode support.
- **Cross-Browser:** Available for Chromium (Chrome, Brave, Edge) and Firefox.
- **Privacy First:** No data collection. Personal API key support for OCR.

## How to Install
To install manually on Chromium browsers:
1. Download and extract the ZIP folder.
2. Open `chrome://extensions/` and enable **Developer Mode**.
3. Click **Load unpacked** and select the extracted folder.
